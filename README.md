# MoonShot

A work of interactive fiction created by Joey & Charity Parrish for the GitHub
Game Off 2020.

https://itch.io/jam/game-off-2020

## Synopsis

You play as a NASA intern in the 1960s, scrambling to make Apollo 11 a success.
